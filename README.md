# Tweet_Widget

## 概要
 簡単にツイートをするためのウィジェット
 
## 機能
 - ツイート本文の入力欄の開閉
 - ツイートしても初期化されないハッシュタグ入力欄
 - ハッシュタグ入力の On/Off を切り替えるチェックボックス
 - ウィジェットの縁を右クリックで表示される 「常に最前面に表示」の On/Off を切り替えるラジオボタン
 - ウィジェットの縁でスクロールすることで不透明度変更
